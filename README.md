# Movie Library

**Автор:** [Гренко Светлана Михайловна]

**Описание:**
Графическое приложение для ведения личной кинотеки. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Реализована фильтрация по жанру и году. Все данные сохраняются и загружаются из файла `data.json` в формате JSON.

**Функционал:**
- Добавление новых фильмов с проверкой корректности ввода.
- Отображение списка фильмов в таблице.
- Фильтрация по жанру и/или году выпуска.
- Сохранение и загрузка данных через JSON.
- Простой и интуитивно понятный интерфейс.

**Примеры использования:**
1. Запустите `movie_library.py`.
2. Введите данные о фильме и нажмите «Добавить фильм».
3. Используйте поля фильтрации для поиска по жанру или году.
4. Сохраните библиотеку кнопкой «Сохранить в JSON».
5. Загрузите ранее сохранённые данные кнопкой «Загрузить из JSON».
